# uic

## Project Setup

```sh
yarn install
```

### Compile and Hot-Reload for Development

```sh
yarn dev
```

### Type-Check, Compile and Minify for Production

```sh
yarn build
```

### Turn on tailwind watcher to compile css
```sh
npx tailwindcss -i ./src/assets/styles/tailwind.css -o ./dist/tailwind.css --watch
```
